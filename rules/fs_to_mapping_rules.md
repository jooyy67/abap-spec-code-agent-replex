# FS → 개발 매핑 입력서 매핑 규칙

## 0. 공통 규칙

| 항목 | 내용 |
|------|------|
| 기능 확장 규칙 | 신규 기능은 기능군(SCR/ALV/EVT/VAL/SAV/EXT/UI/POP/MSG)으로 분류하고 일련번호 부여 |
| 매핑 운영 규칙 | 기능 추가 시 매핑표 수정 X → 8번, 3번에 행 추가 |
| 블록 확장 규칙 | 기존 BLOCK/TEMPLATE 재사용 우선 |

## 12. 내부 자동 생성 규칙

- TEMPLATE_ID = TPL_{BLOCK_ID}
- Include = prefix 기반 자동 매핑
- 기능 ID = {기능군}_{번호}
- 판단 조건에서 기능 ID 직접 사용하지 않음
