## 8. 세부 기능 리스트 (AI 생성 영역)

FS 전체를 기반으로 자동 생성

| 기능 ID | 기능 명칭 | 상세 정의 | 입력서 연결 | 중요도 | 생성 블록 |
|--------|----------|----------|------------|--------|----------|
| SCR_01 | 조회 조건 제어 | 회사코드 고정 및 필드 제어 | Selection | 상 | S01_MODIF |
| ALV_01 | 마스터/상세 연동 | 클릭 시 상세 조회 | ALV | 상 | F01_EVENT |
| EVT_01 | 실시간 계산 | 수량 변경 시 금액 계산 | ALV | 중 | F01_DATA_CHANGED |
| VAL_01 | 저장 검증 | 필수값 및 선택 체크 | Rule | 상 | F01_CHECK_DATA |
| SAV_01 | 저장 처리 | 데이터 저장 처리 | Save | 상 | F01_SAVE_FLOW |
| SAV_03 | DB 반영 | Z-Table / 표준 문서 반영 | Save | 상 | F01_SAVE_DB_DATA |
| EXT_01 | 엑셀 처리 | 다운로드 및 업로드 | File | 중 | F01_DOWNLOAD / F01_UPLOAD |
| ALV_02 | 합계 처리 | 금액/수량 합계 표시 | ALV | 중 | F01_SORT_SUBTOTAL |
| ALV_03 | 시각 효과 | 색상 / 아이콘 표시 | ALV | 중 | F01_COLOR_ICON |
